# LojaOnline

 Loja Online — Projeto Final (React + TypeScript + Ant Design)

Este projeto é uma aplicação completa de Loja Online, desenvolvida com React, TypeScript, Ant Design, Redux, Context API e integração com a Fake Store API. Foi desenvolvido na responsabilidade da metéria de Tópicos Especiais em Programação.

Ele consolida os conteúdos desenvolvidos ao longo das atividades, incluindo:

- Listagem e gerenciamento de produtos
- Sistema completo de carrinho
- CRUD de clientes
- Consumo de API externa
- Persistência em LocalStorage
- Interface moderna e responsiva com Ant Design

Como Executar o Projeto via terminal (cmd):

- Clone o repositório
    - "git clone https://github.com/seu-usuario/loja-online.git"

- Acesse a pasta lojaOnline
    - "cd lojaOnline"

- Instale as dependências
    - "yarn install"
    - "npm install"

- Rodar o Projeto
    - "npm run dev"
